Metabolic Modelling
  • Genome-scale metabolic models attempt to store all published data regarding metabolites, reactions and genes in a metabolic network for a specific organism.
  • They allow for a holistic view of an organism's metabolism.
  • Using linear optimization we can simulate different metabolic scenarios and confont them to experimental results.
  • These models can be a basis for more complex analysis regarding omics (transcriptomics, fluxomics, genomics), as well as chemical data.
Multimodal Transport Modelling
  • Using the Quetzal python module developed by Systra, we can model and forecast traffic for a given transportation network.

Modelling Chemistry - RDKit
  • RDKit is a C++ and Python library capable of handling structural chemical data (molecules and reactions).
  • Classical methods such as subtructure matching, structure similarity or maximum common substructure are directly applied on large quantities of molecules.
  • This allows for screening of large chemical databases to find molecules with specific properties.
Mass Spectrometry - Exploratory Analysis
  • We can read raw data from mass spectrometry directly within python using pyMZML module.
  • A major challenge of Mass Spectrometry is the identification of m/z values found in a sample.
  • By simulating fragmentation patterns from molecules and comparing them with experimental ones, we can assign provisory names to some of the m/z values found.

Chemocobra
App open_in_new
Demo using chemocobra to compare the central metabolism of Escherichia Coli Wild Type vs PKT, the strain patented by Global Bioenergies©. We can see that this web application allows us to define scenarios and, once the model run, allows for comparing specific reactions, metabolites, and display the results in a metabolic map, using a js/python library called Escher.
Explore and compare metabolism of different species
  • This project aims at facilitating the simulation and exploration of metabolism fluxes in different conditions
  • A metabolic model is composed of metabolites (nodes), reactions (edges), as well as genes.
  • You can model different metabolites states by changing the inputs/exchanges as well as limiting fluxes through specific reactions.
A full-stack open source application
  • Frontend was performed with React, a javascript library for dynamic web applications.
  • Backend uses python, with libraries such as django and django rest framework for web development as well as scientific libraries (cobra for metabolic modelling, RDKit for chemical structures handling, pandas for data treatment, etc...)
  • Deployment is performed with docker and docker-compose in Digital Ocean. Reverse proxy is used for this website using Cloudflare.
An ongoing project
  • The next part of chemocobra will be focused on comparing chemical structures of metabolites within a metabolic model.
  • It might allow to predict promiscous activities for existing enzymes as well as toxicity of existing metabolites.
