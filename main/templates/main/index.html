Modelling Chemistry - RDKit
  • RDKit is a C++ and Python library capable of handling structural chemical data.
  • Classical methods such as subtructure matching, structure similarity or maximum common substructure are directly applied on large quantities of molecules.
  • This allows for screening of large chemical databases to find molecules with specific properties.
Mass Spectrometry - Exploratory Analysis
  • We can read raw data from mass spectrometry directly within python using pyMZML module.
  • A major challenge of Mass Spectrometry is the identification of m/z values found in a sample.
  • By simulating fragmentation patterns from molecules and comparing them with experimental ones, we can assign provisory names to some of the m/z values found.

Metabolic Modelling
  • Genome-scale metabolic models attempt to store all published data regarding metabolites, reactions and genes in a metabolic network for a specific organism.
  • They allow for a holistic view of an organism's metabolism.
  • Using linear optimization we can simulate different metabolic scenarios and confont them to experimental results.
  • These models can be a basis for more complex analysis regarding omics (transcriptomics, fluxomics, genomics), as well as chemical data.
